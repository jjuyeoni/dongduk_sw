## 3.프로젝트 내 역할
1. mail, omniauth gem 을 이용하여 메일 인증 로그인 구현
2. sunspot_solr, sunspot_rails gem 을 이용하여 검색 기능 구현
3. 판매자 글작성과 구매자 리뷰작성에 대한 crud 구현
4. 전체 화면 FE 구성 및 개발
